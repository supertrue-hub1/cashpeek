# Яндекс.Метрика: Быстрый старт

## 1. Получить токен (2 минуты)

Перейдите по ссылке (замените `YOUR_CLIENT_ID`):

```
https://oauth.yandex.ru/authorize?response_type=token&client_id=YOUR_CLIENT_ID
```

Если нет Client ID, создайте приложение на https://oauth.yandex.ru

## 2. Добавить в .env

```env
# ID счетчика (обычно уже есть)
NEXT_PUBLIC_YM_ID="12345678"

# Токен (новое!)
METRICA_TOKEN="y0_AgAAAAA..."
```

## 3. Проверить

1. Перезапустите сервер: `bun dev`
2. Откройте `/admin/analytics`
3. Перейдите на вкладку "Метрика"

## Если не работает

### "METRICA_TOKEN не настроен"
→ Добавьте `METRICA_TOKEN` в `.env`

### "Недостаточно прав"
→ При создании приложения выберите `metrika:read`

### "Нет данных"
→ Проверьте, что ID счётчика правильный

### Пустой график
→ Возможно, за период не было визитов
