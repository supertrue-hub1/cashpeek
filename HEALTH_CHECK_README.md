# 🏥 Health Check System - Быстрый старт

Система мониторинга здоровья сайта МФО агрегатора с ИИ-агентом.

## ✅ Что создано

### База данных
- ✅ Модели в `prisma/schema.prisma`:
  - `HealthCheckPage` — страницы для мониторинга
  - `HealthCheckResult` — результаты проверок
  - `HealthIncident` — инциденты
  - `HealthCheckSetting` — настройки

### API Endpoints
- ✅ `/api/health/status` — общий статус системы
- ✅ `/api/health/pages` — управление страницами
- ✅ `/api/health/check` — выполнение проверок
- ✅ `/api/health/incidents` — управление инцидентами
- ✅ `/api/health/chat` — чат с ИИ-агентом

### UI Компоненты
- ✅ `/admin/health` — страница админки
- ✅ `HealthCheckDashboard` — главный дашборд
- ✅ `HealthPagesList` — список страниц
- ✅ `HealthIncidentsList` — список инцидентов
- ✅ `HealthCheckChat` — чат с ИИ-агентом

### Дополнительно
- ✅ Скрипт инициализации страниц
- ✅ Скрипт тестирования
- ✅ Документация

## 🚀 Запуск

### 1. Применение миграций

```bash
npm run db:push
```

### 2. Инициализация начальных страниц

```bash
npx tsx src/lib/health-check-init.ts
```

### 3. Запуск dev-сервера

```bash
npm run dev
```

### 4. Открытие админки

```
http://localhost:3000/admin/health
```

## 📊 Функционал

### Дашборд
- Общий статус системы
- Uptime за 24 часа
- Среднее время ответа
- Количество активных инцидентов
- Распределение статусов

### Страницы
- Список всех страниц в мониторинге
- Фильтрация по категории и статусу
- Поиск по названию и URL
- Ручная проверка страниц
- Просмотр SSL сертификатов
- Uptime статистика

### Инциденты
- Список всех инцидентов
- Фильтрация по статусу
- Детальная информация
- Закрытие инцидентов с комментарием
- История длительности

### ИИ-агент
- Чат-интерфейс
- Команды: статус, проблемы, проверь, отчёт
- Контекстная информация
- Быстрые кнопки команд

## 🤖 Команды ИИ-агента

```
📊 статус       — общий статус системы
🚨 проблемы     — активные инциденты
🔍 проверь [url] — проверить страницу
📈 отчёт        — статистика и тренды
❓ помощь       — все команды
```

## 🔧 Настройка

### Добавление страницы

```typescript
await fetch('/api/health/pages', {
  method: 'POST',
  body: JSON.stringify({
    url: '/mfo/new-mfo',
    name: 'Новая МФО',
    category: 'mfo',
    priority: 2,
    checkInterval: 180
  })
});
```

### Изменение интервала проверок

```typescript
await prisma.healthCheckPage.update({
  where: { id: 'xxx' },
  data: { checkInterval: 60 } // 1 минута
});
```
